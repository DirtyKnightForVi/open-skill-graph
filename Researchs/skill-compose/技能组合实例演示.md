# Skill Compose 技能组合实例演示

让我们通过一个具体的例子，展示 Skill Compose 如何将用户选中的多个技能进行有机组合。

## 场景设定

假设用户想要创建一个"科研文档分析助手"，需要以下技能：
1. **PDF处理技能** - 读取和分析PDF文档
2. **图像生成技能** - 根据分析结果生成可视化图表
3. **PubMed数据库查询技能** - 查找相关研究论文
4. **数据提取技能** - 从文档中提取关键数据

## 步骤1：用户界面选择技能

用户在 Agent 配置界面选择以下技能：
- `pdf` (PDF处理)
- `imagegen` (图像生成) 
- `pubmed-database` (PubMed查询)
- `data-extractor` (数据提取)

## 步骤2：系统构建技能组合

```json
  "max_turns": 60,

```

系统调用 `get_tools_for_agent()` 函数：

```python

```

- `execute_code()` - 执行Python代码
- `bash()` - 执行shell命令

## 步骤3：用户请求处理

### 用户输入
```
"请帮我分析这个PDF文档中的实验数据，并生成相应的图表，同时查找相关的研究论文"
```

### 智能体执行流程

#### 第1步：分析需求并选择合适的技能
智能体分析用户请求，决定使用哪些技能：
- 需要读取PDF → 使用 `pdf` 技能
- 需要生成图表 → 使用 `imagegen` 技能  
- 需要查找相关论文 → 使用 `pubmed-database` 技能
- 需要提取数据 → 使用 `data-extractor` 技能

#### 第2步：读取PDF文档
```python
# 智能体执行代码
result = execute_code("""
import PyPDF2
import re

# 读取PDF文件
with open('research_paper.pdf', 'rb') as file:
    pdf_reader = PyPDF2.PdfReader(file)
    text = ""
    for page in pdf_reader.pages:
        text += page.extract_text()

print("PDF内容提取完成")
print(f"文档总长度: {len(text)} 字符")
""")
```

#### 第3步：提取结构化数据
```python
# 使用数据提取技能
result = execute_code("""
import pandas as pd
import re

# 从文本中提取实验数据
data_pattern = r'(\w+):\s*([\d.]+)\s*±\s*([\d.]+)'
matches = re.findall(data_pattern, text)

# 构建数据表
df = pd.DataFrame(matches, columns=['Parameter', 'Mean', 'Std'])
print("提取的实验数据:")
print(df)
""")
```

#### 第4步：生成可视化图表
```python
# 使用图像生成技能
result = execute_code("""
import matplotlib.pyplot as plt
import seaborn as sns

# 创建图表
plt.figure(figsize=(10, 6))
sns.barplot(data=df, x='Parameter', y='Mean', yerr=df['Std'].astype(float))
plt.title('实验数据可视化')
plt.xlabel('参数')
plt.ylabel('测量值')
plt.xticks(rotation=45)
plt.tight_layout()
plt.savefig('experiment_chart.png', dpi=300, bbox_inches='tight')
print("图表已生成: experiment_chart.png")
""")
```

#### 第5步：搜索相关研究论文
```python
# 使用PubMed查询技能
result = execute_code("""
import requests
from xml.etree import ElementTree as ET

# 构建PubMed查询
query = "experimental data analysis methodology"
url = f"https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi?db=pubmed&term={query}&retmax=5"

response = requests.get(url)
if response.status_code == 200:
    # 解析XML响应
    root = ET.fromstring(response.content)
    id_list = [id_elem.text for id_elem in root.findall('.//Id')]
    
    # 获取详细信息的URL
    details_url = f"https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esummary.fcgi?db=pubmed&id={','.join(id_list)}"
    details_response = requests.get(details_url)
    
    print("找到的相关研究论文:")
    # 解析并显示论文信息
    for i, paper_id in enumerate(id_list, 1):
        print(f"{i}. PMID: {paper_id}")
```

## 技能组合的有机性体现

### 1. 智能调度
智能体根据任务需求智能选择和调度合适的技能，不是简单的技能堆叠。

### 2. 数据流转
各技能间的数据可以无缝流转：
- PDF提取的文本 → 数据提取器
- 提取的结构化数据 → 图像生成器
- 分析的主题 → PubMed查询器

### 3. 环境统一
所有技能共享统一的工作空间和环境变量，确保协作顺畅。

### 4. 上下文保持
整个过程中，智能体保持对任务目标的完整理解，确保各技能输出符合整体需求。

### 5. 错误处理
当某个技能执行失败时，智能体可以尝试替代方案或调整策略。

## 技术实现亮点

### 动态提示词构建
系统根据选中的技能动态构建提示词，确保LLM了解所有可用能力。

### 工具函数绑定
每个技能都转换为标准的工具函数，通过统一的接口调用。

### 执行环境隔离
技能在独立的工作空间中执行，避免相互干扰，同时支持文件共享。

### 环境变量管理
自动注入各技能所需的环境变量，简化配置过程。

这个例子展示了 Skill Compose 如何将多个技能有机地组合成一个完整的解决方案，而不是简单的功能叠加。系统通过智能调度、数据流转和统一环境，实现了真正的技能协作。
